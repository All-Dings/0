# Average-Function

I am a [Mathematics-Function](13000011.md).

## Formula

$$ Average(x_1,x_2 ..\:x_n) = \frac{\sum\limits_{i=1}^{n}{x_i}}{n} $$
$$ = \frac{x_1 + x_2 + .. + x_n}{n} $$

## Python

```python
def Average(X1_X2_Xn_List):
	List_Count = len(X1_X2_Xn_List)
	Sum = 0
	for Xi in X1_X2_Xn_List:
		Sum += Xi
	return Sum / List_Count
```
