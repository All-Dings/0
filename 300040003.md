# Dings_Sound

A Dings_Sound is a [Dings-Object](300040000.md) that can play a [Sound](40200001.md).

## Example

The Sound-File `400000016.mp3` can be included in a Markdown-File with ...

```markdown
![Don't worry, be happy!](400000016.mp3)
```

... and then creates this rendered Output:

![Don't worry, be happy!](400000016.mp3)
